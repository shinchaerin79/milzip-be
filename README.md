# 🪖 MIL.ZIP (밀집)
**군 장병의 모든 혜택을 한곳에, 군장병 통합 복지 플랫폼**

공공데이터 기반 혜택 통합 · 위치 기반 추천 · AI 맞춤 코스 제안

<br/>

2026.04.01 ~ 2026.06.24

<br>

### 🏆 프로젝트 수상 내역
**[헥토그룹 & 서경SW아카데미 프로젝트 성과 발표회] 골드상(1등상) 수상**
<br><br>

### 💡 프로젝트 요약

**해결하고자 하는 문제**
> 군장병 혜택 정보는 병무청·지자체·국군복지포털 등 **여러 플랫폼에 분산**되어 있어 실제 활용이 어렵습니다.

> 외출·휴가 시간이 제한된 군장병은 **지금 바로 이용 가능한 혜택**을 빠르게 찾아야 하지만,  
> 정보의 신뢰성과 최신성을 확인하기 어렵습니다.

<br>

**서비스 개요**

> **MIL.ZIP**은  
> 공공데이터 기반 혜택 정보를 통합하고,  
> AI 맞춤 추천과 위치 기반 탐색으로  
> 군장병이 **제한된 시간 안에 최적의 혜택을 활용**할 수 있도록 설계된 플랫폼입니다.

**주요 기능**
- [x] 공공데이터 기반 군인 할인·복지 정보 통합 제공
- [x] 현재 위치 기반 군인 할인 매장 추천
- [x] AI 맞춤 코스 추천 (동행 유형·카테고리·자유 텍스트 분석)
- [x] CODEF + 카카오 인증 기반 군장병 신원 확인
- [x] 영수증 OCR 기반 실방문 인증 리뷰
- [x] TMO·영화관·청년정책 등 군 복지 정보 통합 제공

<br>

**활용한 핵심 기술**

**[Backend]**  
`Java` · `Spring Boot` · `PostgreSQL` 

**[AI]**  
`OpenAI API` · `pgvector`

**[Infra]**  
`Docker` · `AWS EC2`

**[ETL]**  
`Python` · `FastAPI` · `Selenium`

**[External APIs]**  
`Kakao Mobility API` · `Kakao Local API` · `CODEF API` · `Naver Clova OCR`

<br>

**기대 효과**
> 분산된 군 장병 복지 정보를 하나로 통합하여 혜택 접근성을 높이고,
> 지역 상권에는 신규 고객 유입을, 공공기관에는 공공데이터 활용도 향상을 제공하는
> **군장병·지역 상권·공공기관이 함께 성장하는 선순환 구조**를 구축합니다.

**🪴 Branch Convention (Simple GitHub Flow)**

- `main`: 배포 가능한 브랜치이자 개발 완료된 기능들이 최종적으로 반영되는 브랜치
- `feature/{description}`: 기능 개발 브랜치
    - 예: `feature/social-login`
- `infra/{description}`: 인프라 작업 브랜치
    - 예: `infra/nginx-setting`
